# Andres_Portfolio
Engineering Portfolio
## 🚀 Live Site
https://andresgonzales-hub.github.io/Andres_Portfolio/index.html
## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Vanilla, custom variables, CSS Grid/Flexbox)
* **Design System:** Custom off-white aesthetic (`#fbfdff`), system-first typography, fully responsive.
* **Performance:** Optimized asset loading, zero heavy framework bloat.

## 📂 Project Structure
* `index.html` - Landing page with 3D globe visualization
* `experience.html` - Chronological deep dives into internships (Apple, Boeing) and Capstone projects.
* `about.html` - Background, STEM advocacy, and Real Estate ventures.
* `contact.html` - Direct contact information and professional links.
* `/css/style.css` - Global stylesheet.
* `/assets/` - Optimized project renders, headshots, and graphics.
